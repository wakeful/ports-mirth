# Mirth Connect FreeBSD Port

[Mirth Connect](https://www.mirth.com) - cross-platform HL7 interface engine.

## Usage
```
fetch --no-verify-peer \
  http://github.com/wakeful/ports-mirth/archive/master.zip \
  ; tar -xvf master.zip \
  ; rm master.zip \
  ; cd ports-mirth-master \
  ; make clean install
```
